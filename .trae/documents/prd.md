## 1. 产品概述

家庭药箱管理 Web 应用，帮助家庭用户集中管理药品信息、追踪库存与效期，避免药品过期浪费和用药安全隐患。

- 解决家庭药品管理混乱、过期遗忘、库存不清等问题
- 目标用户为关注家庭健康的普通家庭成员

## 2. 核心功能

### 2.1 用户角色

无需角色区分，所有用户均可使用全部功能。

### 2.2 功能模块

1. **仪表盘页面**：药品总览、即将过期提醒、低库存提醒、快捷操作入口
2. **药品列表页面**：药品卡片/表格展示、筛选与搜索、批量操作

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 仪表盘页面 | 统计概览 | 显示药品总数、即将过期数量、低库存数量、已过期数量 |
| 仪表盘页面 | 即将过期提醒 | 列出30天内即将过期的药品，按过期日期升序排列 |
| 仪表盘页面 | 低库存提醒 | 列出库存≤3的药品，支持一键补货 |
| 仪表盘页面 | 快捷操作 | 新增药品、查看全部药品的快捷按钮 |
| 药品列表页面 | 搜索筛选栏 | 支持按症状关键词搜索、按过期状态筛选、按库存状态筛选 |
| 药品列表页面 | 药品卡片列表 | 展示药品名称、症状、库存、过期日期、存放位置，支持编辑/消耗/补货 |
| 药品列表页面 | 新增/编辑弹窗 | 表单录入药品名称、适用症状、库存数量、过期日期、存放位置 |

## 3. 核心流程

用户打开应用 → 仪表盘查看提醒 → 点击药品进行管理 → 新增/编辑/消耗/补货操作 → 数据实时更新

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：医药健康风 — 深青绿 (#0D9488) 为主色，暖珊瑚 (#F97066) 为警示色，浅薄荷底色
- **按钮风格**：圆角药丸形按钮，带微阴影，hover 时上浮
- **字体**：标题使用 Noto Serif SC 衬线体，正文使用 Noto Sans SC 无衬线体
- **布局风格**：左侧固定导航 + 右侧内容区，卡片式布局
- **图标风格**：线性图标搭配柔和色彩圆点背景

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 仪表盘页面 | 统计卡片 | 4张圆角卡片横排，图标+数字+描述，渐变背景色 |
| 仪表盘页面 | 提醒列表 | 带时间线图标的列表，过期红色标记，低库存橙色标记 |
| 药品列表页面 | 搜索筛选栏 | 搜索输入框+下拉筛选+状态标签切换 |
| 药品列表页面 | 药品卡片 | 圆角卡片，左上角状态色条，信息分层排列 |
| 药品列表页面 | 操作弹窗 | 居中模态框，毛玻璃背景，表单字段竖排 |

### 4.3 响应式设计

桌面优先设计，平板和移动端自适应：
- 桌面：侧边导航 + 多列卡片
- 平板：顶部导航 + 两列卡片
- 移动端：底部导航 + 单列卡片
